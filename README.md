Taller_Colorea

Landing page tipo one-page desarrollada con HTML y CSS, inspirada en arte digital y el uso de inteligencia artificial.

Descripción
Este proyecto presenta una interfaz creativa donde se combina el dibujo tradicional con un enfoque tecnológico, utilizando una ilustración tipo conejo como elemento principal.

Tecnologías usadas
- HTML5
- CSS3
- Inteligencia Artificial (apoyo en diseño y estructura)

Características
- Diseño moderno y juvenil
- Secciones: Header, Body y Footer
- Uso de imagen sin fondo
- Estilo visual con temática digital (binario)
